hijax
=======
Hijax is a no-refresh JavaScript library for creating Native-Like application.

Consist
=======
Hijax is consist of seven parts: <br />
core(hijax)<br />
ajaxify(hijax.ajaxify), 
module(hijax.module),
cache(hijax.cache), 
request(hijax.request), 
result(hijax.result), 
template(hijax.template)
